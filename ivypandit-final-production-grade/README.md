# Ivy Pandit Website

Production-ready Next.js website for **Ivy Pandit**.

## Deploy on Vercel
1. Upload these files to the root of your GitHub repository.
2. Import the repository in Vercel.
3. Use the default Next.js settings.
4. Add `ivypandit.com` and `www.ivypandit.com` under Project Settings → Domains.

## Local development
```bash
npm install
npm run dev
```
