# Ivy Pandit Website

Production-ready Next.js website for Ivy Pandit.

## Deploy
Upload the contents of this folder to GitHub root and let Vercel deploy automatically.

## Required structure
- app/page.tsx
- app/layout.tsx
- app/globals.css
- public/images/ivy-pandit-homepage-hero.png
- package.json
- next.config.mjs
- tsconfig.json
